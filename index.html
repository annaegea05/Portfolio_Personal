Proyectos

Travel Dashboard

Dashboard de viajes con HTML, CSS y JavaScript vanilla para consultar el tiempo, la probabilidad de lluvia y la moneda local de una ciudad antes de viajar. Carga datos en tiempo real desde APIs externas con fetch, incluye un selector de ciudad, un conversor de euros a moneda local y un mensaje de recomendación según los datos. Diseño responsive con metodología Mobile First.
